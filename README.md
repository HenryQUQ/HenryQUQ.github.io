# Chenyuan Qu Personal Website

An academic personal website for Chenyuan Qu, focused on research, publications, datasets, and selected project work.

![Homepage preview](docs/readme-preview.png)

Live site: [chenyuanqu.com](https://chenyuanqu.com)

A quiet, editorial single-page site with publication spotlights, dataset highlights, and project-led media.

Built with Next.js, TypeScript, and Tailwind CSS, then statically exported for GitHub Pages.
